# Atlas of Diagnostic Features of Chinese Ceramics and Porcelain

A scientific reference system for the identification, typological classification, and materials analysis of Chinese ceramic artifacts.

**Author:** Professor Nona Dronova  
**Method:** Dronova Non-Destructive Optical Authentication — combining gemmological microscopy, ceramic technology, microstructural analysis, and historical research.
