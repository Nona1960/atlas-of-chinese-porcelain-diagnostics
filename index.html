What this system is

A diagnostic reference
instrument, not
an encyclopaedia

The Atlas is structured as a scientific knowledge graph. Every entry is an instance of a typed entity — Reference Object, Diagnostic Feature, Kiln, Dynasty, Material, Publication, or Methodology — with defined required fields, a unique persistent identifier, and explicit bidirectional relationships to all connected entities.

The diagnostic focus distinguishes this system from catalogue databases or museum records. Each Diagnostic Feature entry documents not merely the visual appearance of a property but its attribution significance: what its presence indicates, which kilns and periods it is associated with, and by which analytical method it is identified.

Knowledge is added in discrete, validated units. Attribution certainty is classified at four levels and displayed at every cross-link. No claim in the system is undocumented.

Analytical framework

Diagnostic methodology

Each diagnostic feature in the atlas is identified by at least one documented analytical method. Four categories of analysis form the core diagnostic framework.

MTH-SEM · MTH-PLM

Microstructure Analysis

Examination of ceramic body and glaze at the microscopic scale. Cross-section preparation followed by polarised light microscopy (PLM) or scanning electron microscopy (SEM) reveals inclusion density, vitrification degree, glaze–body interface character, and firing temperature indicators.

Technique SEM-BSE / PLM / OM
Scale 10 μm – 1 mm
Sampling Micro-destructive
Output Texture · Porosity · Phase

MTH-XRF · MTH-SEM-EDX

Glaze Composition Analysis

Elemental analysis of glaze layers by X-ray fluorescence (XRF) or energy-dispersive X-ray spectroscopy (EDX). Quantifies major and minor oxide concentrations (SiO₂, Al₂O₃, CaO, Fe₂O₃, etc.) enabling kiln attribution by chemical signature.

Technique XRF / SEM-EDX / PIXE
Precision <0.1 wt% (major oxides)
Sampling Non-destructive / micro
Output Oxide wt% composition

MTH-VIS · MTH-PLM

Crackle Pattern Analysis

Systematic visual and microscopic characterisation of crazing networks in ceramic glazes. Pattern morphology, network density, crack width, and depth of penetration are diagnostic for specific kiln traditions. Guan, Ge, and Ru wares exhibit taxonomically distinct crackle signatures.

Technique Macro visual / PLM
Parameters Density · Width · Depth
Sampling Non-destructive
Output Pattern taxonomy

MTH-VIS · MTH-XRD

Firing Trace Analysis

Identification and classification of marks left by kiln furniture and atmospheric conditions during firing. Spur marks, sand adhesions, sagging deformations, and reduction or oxidation surface effects are diagnostic for specific firing technologies and kiln types. Phase analysis by XRD confirms firing temperatures.

Technique Visual / XRD / SEM
Temp. range 950°C – 1350°C
Sampling Non-destructive primary
Output Atmosphere · Support type

Information architecture

A knowledge graph,
not a website

Navigation in the Atlas is not hierarchical descent through page levels. It is traversal of a typed knowledge graph. Every entity page exposes all its relationships explicitly. From any page, the user can move to any linked entity in any domain.

I

Entity-first structure

Every page is an instance of a typed entity (RO, DF, KLN, DYN, MTH, PUB, MAT). Type determines required fields, URL format, and relationship set.

II

Bidirectional cross-links

All relationships are bidirectional. If Object A links to Feature X, Feature X lists Object A. Enforced at data layer. No manual maintenance.

III

Attribution certainty at every link

Kiln attributions display certainty level at the link — not only on the object page. Four levels: Confirmed · Probable · Proposed · Disputed.

IV

Permanent, stable URLs

URLs are defined by entity type and identifier. Attribution revisions do not change URLs. Redirects are logged centrally. All links remain valid indefinitely.
