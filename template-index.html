Scientific Page
Template System

8 reusable structural templates. Each template is an entity type in the knowledge graph. Every page in the Atlas is an instance of one of these templates. No new templates may be created without a versioned revision to this specification.

Template Library

→ 01 ART · Scientific Article

Scientific
Article

Long-form analytical studies: glaze analysis, crackle analysis, microstructure research, diagnostic frameworks, comparative studies. Full peer-review structure with image documentation.

Abstract Introduction Historical context Material analysis Macroscopic obs. Microscopic analysis Diagnostic criteria Comparative analysis Forgery detection Image gallery Bibliography
→ 02 RO · Reference Object

Reference
Object

Individual ceramic artifact records. Primary data entry: the empirical core of the Atlas. Full physical, analytical, and interpretive documentation with mandatory image set.

Object ID Image documentation Physical description Provenance Material composition Glaze analysis Microstructure Diagnostic interpretation Attribution certainty
→ 03 DF · Diagnostic Feature

Diagnostic
Feature

Observable or measurable properties with attribution value. Categories: glaze, microstructure, firing traces, form morphology, decoration, material. Includes forgery detection notes.

Definition Formation mechanism Visual characteristics Microscopic indicators Diagnostic significance Misinterpretations Forgery detection Distribution by period/kiln Classification position
→ 04 MTH · Methodology

Analytical
Methodology

Technical reference for analytical and examination techniques. Operating principles, instrumentation, step-by-step workflow, limitations, and diagnostic decision logic.

Method overview Scientific principles Workflow (5 steps) Glaze application Microstructure application Limitations Decision flow Case studies
→ 05 PUB · Publication

Scholarly
Publication

Structured bibliographic records for all works cited in the Atlas. Covers articles, monographs, excavation reports, exhibition catalogues. Links to all documented Atlas entities.

Citation block Metadata grid DOI / stable URL Abstract Methodology Results Discussion Atlas entities documented
→ 06 DYN · Dynasty / Period

Dynasty
/ Period

Chronological scaffolding of the Atlas. Structured index pages, not art-historical narratives. Active kilns, characteristic features, material properties, and representative objects per period.

Timeline structure Ceramic history (200w max) Kiln development Material characteristics Diagnostic relevance Reference objects Prev / Next dynasty
→ 07 KLN · Kiln

Production
Kiln

Ceramic production sites as knowledge entities. Geographic location, active periods, production techniques, material signatures, typical defects, and attributed objects with certainty levels.

Geographic context Activity timeline Production techniques Material characteristics Defects / variations Kiln technology Attributed objects Glaze systems
→ 08 AUT · Author Profile

Author
Profile

Researcher records. Academic identity, ORCID and external identifiers, research focus, methodological contributions, publication list, and Atlas entity contributions.

Academic identity ORCID / identifiers Research focus Methodological contribution Publications list Atlas role Co-authors

Navigation Logic

Breadcrumb logic (canonical path)

Atlas › Objects › [Dynasty] › [Kiln] › [RO-ID] Atlas › Features › [Category] › [Feature name] Atlas › Kilns › [Region] › [Kiln name] Atlas › Periods › [Dynasty] › [Sub-period] Atlas › Methodology › [Category] › [Method] Atlas › Publications › [PUB-ID] Atlas › Publications › Authors › [AUT-ID]

Breadcrumb reflects canonical URL path, not session history. Current page item is not a link. Mobile: collapses to parent + current.

Sidebar panel structure (all entity pages)

1 — On this page (anchor links to H2 sections)
2 — Related entities of type A (with IDs)
3 — Related entities of type B (with IDs)
4 — Related entities of type C (with IDs)
5 — Key publications

Entity type sequence in sidebar varies by template type. Relationship panel is always complete — all linked entities displayed, not just those mentioned in body text.

Cross-link display rule

Every cross-link displays: entity name + entity ID. Attribution certainty is shown at every RO→KLN link (not only on the RO page). Links use the .sidebar-link component from templates.css. Orphan pages (no cross-links) are a data quality error.

URL permanence rule

URLs are defined by entity type + identifier. If an attribution changes, the URL does not change. Revisions are documented in config/redirects.json. URLs must remain valid for the lifetime of the Atlas.
